# MA0005 - Use Array.Empty<T>()

You should use `string.Equals` instead of `==` or `!=` to improve the readability of the code.

````csharp
"" == "";

// Should be
string.Equals("", "", StringComparison.Ordinal);
````

````csharp
"" != "";

// Should be
!string.Equals("", "", StringComparison.Ordinal);
````
